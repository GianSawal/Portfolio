Enterprise Network Services Deployment

This project demonstrates the deployment of essential enterprise network services including DHCP, DNS, and wireless infrastructure to support both wired and wireless client connectivity within a small network environment.

Network Overview

The network infrastructure connects multiple client devices through a central switching system that allows communication between wired and wireless devices. A dedicated server is deployed to provide essential network services such as DNS resolution.

Dynamic Host Configuration Protocol (DHCP) is used to automatically assign IP addresses and network settings to devices joining the network. This allows computers and laptops to connect without requiring manual IP configuration.

Wireless connectivity is provided through a router and an access point, allowing mobile devices and laptops to access the network while maintaining communication with wired devices connected through the switch.

This project demonstrates how enterprise network services such as DNS, DHCP, switching, and wireless access work together to provide reliable connectivity within an organizational network.

Technologies Used

DHCP

Automatically assigns IP addresses and network settings to devices connecting to the network.

DNS Server

Provides domain name resolution allowing devices to locate network services.

Switching Infrastructure

Connects network devices and manages traffic flow between clients and servers.

Wireless Networking

Allows laptops and mobile devices to connect to the network without physical Ethernet connections.

Gateway Router

Provides network routing and acts as the gateway for devices accessing external networks.

IP Addressing

Uses the 192.168.1.0/24 network to organize device communication.
